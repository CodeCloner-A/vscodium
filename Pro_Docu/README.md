# Dokumentation – VSCodium-Fork mit integriertem KI-Agenten

Dieses Verzeichnis enthält die vollständige Dokumentation des Projekts: ein VSCodium-Fork („eigene IDE"), in dessen Core ein KI-Agent auf Basis von Firebase AI Logic (Gemini) integriert ist.

Die Dokumentation folgt dem in der professionellen Anwendungsentwicklung etablierten **Diátaxis-Framework**, das Inhalte nach ihrem Zweck trennt: *Tutorials* (Lernen), *How-to-Anleitungen* (Erledigen), *Referenz* (Nachschlagen) und *Erklärung* (Verstehen).
